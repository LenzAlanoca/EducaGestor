Declaracion De Propositos

El propósito del Sistema de Seguimiento Académico Virtual es facilitar una plataforma integral y eficiente para la gestión y seguimiento académico en instituciones educativas. Este sistema se desarrolla con el objetivo de proporcionar herramientas y recursos que mejoren la comunicación, organización y transparencia dentro del entorno educativo, beneficiando a usuarios clave como estudiantes, profesores, administradores y personal directivo.
El sistema permitirá:

  · Registrar y mantener datos actualizados de usuarios (estudiantes, profesores, administradores) incluyendo información personal y roles de acceso.
 · Permitir a los administradores crear y configurar nuevas cuentas de usuario según las necesidades institucionales.

 · Facilitar a los usuarios el acceso a información relevante sobre su desempeño académico, incluyendo notas, horarios y comunicaciones institucionales.
 · Proporcionar informes detallados sobre la actividad de usuarios, permitiendo a los administradores delimitar y supervisar el acceso según las normativas establecidas.

 · Brindar a los estudiantes información detallada sobre asignaturas, horarios de clases y profesores asignados.
 · Permitir a los profesores acceder a información sobre sus asignaturas y la lista de estudiantes inscritos.

 · Servir como medio para la difusión de normas, reglamentos y disposiciones institucionales desde la dirección hacia la comunidad educativa.
 · Generar informes detallados sobre la inscripción a las materias y otros datos relevantes para la dirección.

Lista de Acontesimientos

 · Un usuario solicita acceso al sistema proporcionando sus datos personales y el tipo de acceso requerido (estudiante, profesor, administrador).

 · Un administrador utiliza el sistema para crear una nueva cuenta de usuario, ingresando los datos necesarios y configurando los permisos de acceso correspondientes.

 · Un estudiante accede al sistema para ver su información académica, como notas, horarios de clases y detalles del curso.

 · El sistema genera un informe detallado sobre la actividad de un usuario, incluyendo registros de acceso y acciones realizadas.

 · El sistema asigna automáticamente profesores a cursos y genera horarios de clases para cada estudiante.

 · El sistema envía informes a la dirección sobre la cantidad de estudiantes inscritos en cada materia y grado.

 · La dirección utiliza el sistema para difundir normas, reglamentos y disposiciones a través de mensajes o anuncios dentro de la plataforma.

 · El sistema genera un informe detallado sobre la actividad de un usuario, incluyendo registros de acceso y acciones realizadas.

 · El sistema solicita confirmación de acceso a un usuario nuevo por parte de un administrador antes de que la cuenta sea activada.

 · Un administrador restringe o amplía los permisos de acceso de un usuario en base a políticas institucionales o cambios en el rol del usuario.
